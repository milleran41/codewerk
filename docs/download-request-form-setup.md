# CodeWerk: форма запроса ссылки

Эта форма нужна для крупных программ, где мы не даём прямое скачивание сразу.
Например: Taste & Trace / Кулинарная книга и MixLab.

## 1. Создать Google Form

Название формы:

`CodeWerk download request`

Описание:

`Оставьте email, и мы отправим ссылку на скачивание выбранной программы CodeWerk.`

Поля:

1. `Email`
   Тип: электронная почта или короткий ответ с проверкой email.
   Обязательное поле.

2. `Program`
   Тип: раскрывающийся список или один вариант из списка.
   Обязательное поле.

Варианты:

- `Taste & Trace / Кулинарная книга`
- `MixLab`

Важно: названия вариантов должны совпадать с названиями в файле
`docs/download-request-apps-script.js`.

## 2. Привязать таблицу

В Google Form откройте вкладку `Ответы` и нажмите значок Google Sheets.
Создайте новую таблицу для ответов.

## 3. Добавить автоответчик

В созданной Google Таблице откройте:

`Расширения -> Apps Script`

Удалите пустой код и вставьте содержимое файла:

`docs/download-request-apps-script.js`

Сохраните проект.

## 4. Включить запуск при отправке формы

В Apps Script откройте `Triggers` / `Триггеры`.

Создайте триггер:

- Function: `onFormSubmit`
- Event source: `From spreadsheet`
- Event type: `On form submit`

При первом запуске Google попросит разрешения на отправку email.
Нужно разрешить, иначе письма не будут уходить.

## 5. Проверить

Откройте форму как пользователь.
Выберите программу.
Введите свой email.
Отправьте форму.

Через несколько секунд должно прийти письмо со ссылкой на скачивание.

## 6. Подключить форму к сайту

После создания формы скопируйте публичную ссылку вида:

`https://docs.google.com/forms/d/e/.../viewform`

Её нужно вставить в `script.js` как отдельную ссылку для кнопки
`Получить ссылку`.

Если нужна предварительная подстановка программы, используйте ссылку
`Получить предварительно заполненную ссылку` в Google Form.
