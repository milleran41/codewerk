# CodeWerk

Статический одностраничный сайт-портфолио для программ Andreas Miller, опубликованных на GitHub.

## Как добавить новую программу

1. Откройте `data/projects.json`.
2. Скопируйте один объект проекта и замените:
   - `id` - уникальный якорь, например `calendar-germany`;
   - `title` - название программы;
   - `status` - поставьте `published`, чтобы карточка появилась на сайте;
   - `description` - описание из README проекта;
   - `features` - список основных возможностей;
   - `screenshot` - путь к локальному скриншоту в `assets/screenshots/` или raw-ссылка на изображение из GitHub README;
   - `githubUrl` - ссылка на GitHub-репозиторий;
   - `downloadUrl` - прямая ссылка на файл из GitHub Releases или ссылка на Releases;
   - `qrImage` - путь к QR-коду в `assets/qr/`;
   - `qrTarget` - ссылка, на которую должен вести QR-код.

Проекты со статусом `soon` не показываются в разделе “Мои программы”.

Сейчас на сайте опубликованы: Taste & Trace, Timer, MixLab, Kalender Deutschland, Linkora и LinkVault. `linkora-v2` не добавлен отдельно, потому что его README указывает на основной репозиторий `linkora`. `Bewerbungsunterlagen` не добавлен, потому что это личное портфолио, а не отдельная программа.

## Как создать QR-код

Для карточки программы QR-код должен вести не на скачивание, а на якорь этой программы на сайте:

```text
https://milleran41.github.io/codewerk/#cookbook
```

Так человек сможет открыть страницу программы на телефоне, прочитать описание и позже скачать файл уже на компьютере. Не делайте QR-коды на прямые ссылки `.exe` или `.zip`: телефон может сразу начать скачивать компьютерную программу.

Например, QR для таймера должен вести на:

```text
https://milleran41.github.io/codewerk/#timer
```

Если сайт ещё не опубликован на GitHub Pages, ссылка `https://milleran41.github.io/...` будет открывать 404. Сначала опубликуйте сайт, проверьте адрес в браузере, и только потом создавайте QR-коды.

Для главной визитки QR-код должен вести на главную страницу:

```text
https://milleran41.github.io/codewerk/
```

Сохраните QR-код в папку `assets/qr/`, например `assets/qr/cookbook.png`, и укажите этот путь в `data/projects.json`.

## Как опубликовать на GitHub Pages

1. Загрузите файлы сайта в GitHub-репозиторий.
2. Откройте настройки репозитория: `Settings` -> `Pages`.
3. В разделе `Build and deployment` выберите источник `Deploy from a branch`.
4. Выберите ветку `main` и папку `/root`.
5. Сохраните настройки и дождитесь публикации.

После публикации сайт будет доступен по адресу:

```text
https://milleran41.github.io/codewerk/
```

## Важное про локальный запуск

Сайт можно открыть двойным кликом по `index.html`. На GitHub Pages список программ загружается из `data/projects.json`. При локальном открытии без сервера используется встроенная резервная карточка, потому что браузеры часто блокируют чтение локальных JSON-файлов.
